## 2026-02-27 - [Contextual CTAs in Empty States]
**Learning:** Adding a "Request for this day" button in the empty state of the daily absence details view significantly improves the user flow. It transforms a dead-end into a productive interaction by pre-filling the absence request form with the selected date.
**Action:** Look for similar "dead-end" empty states in detail panels (e.g., project tasks, comments) and provide contextual actions that pre-fill relevant data.

## 2026-03-03 - [Language Cohesion in Localized Views]
**Learning:** In interfaces where multiple languages are used inconsistently (e.g., Italian labels with English buttons), users experience higher cognitive load. Aligning the primary action labels (e.g., "Add task" -> "Aggiungi task") with the surrounding UI language makes the application feel more trustworthy and native.
**Action:** Always verify the language of adjacent UI elements before adding new text, and proactively fix minor inconsistencies in the immediate vicinity to maintain tonal cohesion.
